# The Digital Séance

A tarot-themed scenario exercise exploring how AI could reshape work — drawn as a five-card spread (Automation, Judgment, Jevons, Roles, Memory), each card landing on a low/mid/high outcome.

## Running it

No build step required — open [index.html](index.html) directly in a browser, or serve the folder with any static file server:

```
npx serve .
```

## Deploying

This is a static site (React + Babel loaded from CDN, compiled in-browser). It can be hosted as-is on GitHub Pages: enable Pages on this repo pointing at the `main` branch / root folder.
